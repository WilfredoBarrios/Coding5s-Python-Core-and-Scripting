Welcome to \*\*Stage 5\*\*. This is the ultimate crucible. Here, you are no longer just an island solving small puzzles; you are an architect scaling corporate code to fit realistic corporate demands.



\## 🧠 Cognitive Friction Blueprint

You will be given fully functional, high-level modular scripts. Your friction consists of incoming real-world requirement spikes, simulated via technical Jira tickets. You must scale the logic to handle unexpected edge cases, integrate new diagnostic auditing features, and inject fault tolerance without breaking existing business contracts.



\- \*\*Specialized AI Mentor:\*\* `The Strategic Partner` (Visionary, cut-throat system architect).

\- \*\*Core Strategy:\*\* Defending code robustness, error handling under stress, and production scaling.
