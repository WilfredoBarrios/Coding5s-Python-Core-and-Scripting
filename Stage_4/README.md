Welcome to **Stage 4**. Writing code that works is easy; writing scalable, maintainable, and idiomatic production code is what separates true software engineers from amateurs.

## 🧠 Cognitive Friction Blueprint
This stage forces you to attack technical debt. You will be handed functional but messy, monolithic, and flat scripts. Your job is to enforce strict styling conventions (PEP 8), encapsulate logic into clean modular functions, eliminate hardcoded values, and reduce algorithmic complexity.

- **Specialized AI Mentor:** `The Elegant Critic` (Sarcastic, purist, delivering "tough love").
- **Core Strategy:** Decoupling logic, applying idiomatic Python patterns, and refactoring monolithic execution trees into modular structures.
