## 🛠️ Imaynatataq sapa Livebook nisqata llamk'achiy?

Sapa `.livemd` archivo ukhupi, kay simple workflow nisqata qatipay:

1. Archivo nisqata **[Livebook] (explained as [kawsay qillqa wasi])** nisqapi kichay (desktop utaq browser version).
2. **Stage 1 prompt** nisqata copy ruway, chaymanta preferred AI chat nisqaykiman (ChatGPT, Gemini, Claude, etc.) paste ruway.
3. AI nisqaqa **3-manta 5-kama executable micro‑examples** nisqakunata topic nisqapaq generarunqan.
4. Kay examples nisqakunata Livebook nisqapa **execution cell** nisqanman copy ruway, chaymanta run ruway.
5. [Code] (explained as [qillqa wakichiy]) nisqata modify ruway, experiment ruway, chaymanta output nisqata real time nisqapi tikrakuyta qhaway.

## 🧠 The Socratic Mentor (AI)

Sapa Livebook nisqaqa **[Mentor prompt] (explained as [yanapaq tapuy qillqa wakichiy])** nisqatapas hap'in. Kayta same AI chat nisqaman code nisqaykiwan logic nisqaykipa short explanation nisqanwan paste ruwayta atinki. Mentor nisqaqa:

- **Manam kutichiyta qunqachu directamente** (ñawpaqtaqa)
- Socratic questions nisqakunata tapusunki pantay nisqata kikiyki descubrinaykipaq
- Real‑world analogies nisqakunata **[programming concepts] (explained as [qillqa wakichiy yuyaykuna])** nisqakunata explicanaykipaq llamk'achin
- "Vibe Coding" nisqata chinkachiyta yanapasunki, chaymanta true engineering intuition nisqata ruwanaykipaq
- Senior‑level answers nisqakunata qun specific questions nisqakunata tapuspa

⚠️ CRITICAL INFRASTRUCTURE NOTE FOR MAYAN OR NATIVE LANGUAGES: Maximum language insulation nisqata, robust inline comment logic nisqata, chaymanta precise dynamic glossaries nisqata localized versions nisqapi (Mayan languages nisqakunatapas hap'ispa) aseguraypaq, Gemini Pro (utaq aswan hatun) nisqapa llamk'achiyninmi ancha recomendado. Free-tier models nisqakunaqa manam syntactic attention nisqayuqchu kay lenguajes nisqakunata procesanaypaq, paykunaqa limited training data nisqayuqmi AI models nisqapaq. America Continentepi Mayan chaymanta native languages nisqakunapaq, context seed nisqam prompts ukhupi embedasqa kachkan outputs nisqapa calidad nisqanta maximizaypaq; chayraykum, premium model nisqata recomendanku mejores resultados nisqakunata chaskinaykipaq.

## 🏛️ Architecture & Mentors

Sapa lab nisqaqa **[Explanation Gate] (explained as [qhawachiy punku])** nisqata hap'in, maypichus specialized AI Mentor nisqa yachay process nisqaykita pusasunki. Sapa stage nisqapaq mentor personality nisqanmi kachkan, phase nisqapa psychological pedagogical needs nisqanwan match ruwanaypaq diseñadasqa:

| Stage | Mentor | Personality | What It Teaches |
|-------|--------|-------------|------------------|
| **1** | The Safe Guide | Warm, patient, like an older sibling | Safe [syntax] (explained as [qillqa kamachiy]) exploration, "vibe coding" nisqata chinkachiy |
| **2** | The Analytical Partner | Curious, coffee‑fueled "log detective" | [Error] (explained as [pantay]) traces nisqata ñawiy, crashes nisqakunata entiendiy |
| **3** | The Logical Scaffold | Methodical, encouraging "senior builder" | Data flow nisqata visualizay (Input → Transform → Output) |
| **4** | The Elegant Critic | Sarcastic, purist, "tough love" | Idiomatic [code] (explained as [qillqa wakichiy]), functional patterns, malos hábitos nisqakunata chinkachiy |
| **5** | The Strategic Partner | Visionary, professional "system architect" | Edge cases, fault tolerance, business logic alignment nisqakuna |
