Welcome to **Stage 1**. This is the entry point of the Coding5s ecosystem, engineered to build baseline muscle memory and completely eradicate "vibe coding" and passive consumption.

## 🧠 Cognitive Friction Blueprint
In this stage, you are introduced to the core concepts using the **[NO KEYBOARD RULE]**. You are strictly prohibited from copying and pasting code. Your goal is to manually type out every exercise inside Livebook, interacting directly with runtime outputs to internalize raw syntax.

- **Specialized AI Mentor:** `The Safe Guide` (Warm, highly patient, acting like an older sibling).
- **Core Strategy:** Flat scripts, direct execution, and foundational syntax modeling without modular abstractions.
