Welcome to \*\*Stage 3\*\*. This lab focuses heavily on pipeline mapping and architectural context, simulating a common engineering scenario: jumping into an existing, incomplete data system.



\## 🧠 Cognitive Friction Blueprint

You will find functional scripts that are missing core engine components, marked clearly by `# TODO` blocks or broken pipelines. Your goal is to read the macro architecture, visualize data streams, and complete the missing logical steps without disrupting the surrounding code parameters.



\- \*\*Specialized AI Mentor:\*\* `The Logical Scaffold` (Methodical, encouraging senior builder).

\- \*\*Core Strategy:\*\* Mapping Input ➔ Transform ➔ Output, respecting established interfaces, and finishing data loops.
